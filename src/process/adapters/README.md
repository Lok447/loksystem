# Process Adapters

This directory is reserved for transport and platform adapters that sit outside
the core business modules.

- `http/`: HTTP route/controller adapters
- `websocket/`: event-stream / socket adapters
- `electron/`: Electron-only host adapters
- `standalone/`: standalone server/process bootstrap adapters
