# 最小发版回归清单

每次发版前固定执行以下 6 项。

## 1. 启动
- 应用可启动，不依赖旧盘符路径
- 图标、启动页、About 页、Lok 品牌文案正确
- 默认语言符合预期

## 2. 会话
- 能新建会话
- 能上传附件
- Lok CLI / Hermes 能读取附件并生成 workspace 文件
- 桌面端能看到新文件

## 3. 助手
- 预置助手可用
- 旧用户升级后，自定义助手不丢失
- 语音输入和常用工具配置能正常保存

## 4. 团队
- leader 能派任务给至少 2 个成员
- 一个成员完成、一个成员失败时，leader 能收到汇总
- 失败成员的未完成任务不会永久挂死

## 5. WebUI
- WebUI 能看到同一会话与 workspace 变化
- 静态官网页可打开
- About 页按钮跳转和离线打开表现正常

## 6. 打包
- 至少执行一次 `npm run package`
- Windows x64 安装包命名正确
- `latest*.yml` 等元数据齐全
- 产物中不再依赖旧 `C:` 路径
